# hush-userscripts

A collection of browser userscripts.

## Scripts

### Youtube Vidya Fixer (`youtube-vidya-fixer/`)

Removes YouTube's playback speed memory and forces a fixed speed. Press `H` to increase, `G` to decrease by 0.25x.

- **Install**: Install via Tampermonkey/Greasemonkey, or copy the `.user.js` file.
- **Match**: `https://www.youtube.com/*`
